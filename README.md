# Future Is Now - Static Website

This repository contains a fully static, responsive website for Future Is Now. It runs directly from `index.html` and can be hosted on GitHub Pages.

## Structure
- `index.html` - Main one-page site
- `product-attendance.html` - Optional product page
- `styles.css` - Theme and layout
- `script.js` - Menu, animations, and contact form mailto
- `assets/logo.png` - Company logo
- `assets/og-image.png` - OpenGraph image

## Run locally
Open `index.html` in a browser.

## GitHub Pages
1. Push the repo to GitHub.
2. Go to Settings > Pages.
3. Set Source to `Deploy from a branch`.
4. Select the `main` branch and `/ (root)` folder.
5. Save. Your site will be available at the Pages URL.
