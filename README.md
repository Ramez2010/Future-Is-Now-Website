# Future Is Now - Static Website

Static HTML/CSS/JS website for Future Is Now. Compatible with direct file open and GitHub Pages.

## Pages
- `index.html`
- `product-attendance.html`
- `pricing.html`
- `contact.html`
- `privacy.html`
- `terms.html`
- `refund.html`

## Assets
- `assets/logo.png`
- `assets/og-image.png`

## Run locally
Open `index.html` in a browser.

## GitHub Pages
1. Push to GitHub.
2. Settings > Pages.
3. Deploy from branch: `main` and `/ (root)`.
4. Save.
