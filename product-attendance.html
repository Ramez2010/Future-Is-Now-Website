Odoo Attendance Mobile App

Product Overview

Enable accurate, real-time attendance tracking across offices and field teams.

Fast check-ins

Capture check-in/out with required notes and optional GPS validation.

Odoo-native sync

Push records directly to Odoo (attendance, notes, GPS, attachments).

Secure login

Device binding to keep each employee account tied to a single device.

Key Features

Current capabilities included in the FIN Attendance solution.

Project selection

Select the analytic account (project) during check-in and check-out.

GPS options

Capture GPS coordinates, accuracy, and address for check-in/out.

Geofence per project

Set multiple locations and radius per project (analytic account) to prevent out-of-site check-ins.

Selfie verification

Optional selfie on check-in and/or check-out per employee or per project policy.

Manager notifications

Managers receive inbox messages (and Android push if enabled) on check-in/out.

Inbox messages

Send announcements and scheduled/repeat messages to mobile app employees.

Photos before/after work

Attach multiple images for job proof; download or bulk-delete from Odoo.

Live map tracking

View checked-in employees on a live map with last-seen timestamps and quick zoom.

Time off (Leave)

Submit leave requests, upload attachments, and track balances from the mobile app.

Payslips (optional)

Let employees view and download payslip PDFs if Payroll is enabled in Odoo.

Bilingual UI

English (LTR) and Arabic (RTL) supported in the mobile app.

Real screens from the mobile app and Odoo backend.
